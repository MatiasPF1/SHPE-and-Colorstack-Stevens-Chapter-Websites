# SHPE and ColorStack Stevens Chapter — Websites
Source code for the official chapter websites of **SHPE Stevens** and **ColorStack Stevens**, The two most important organizations for Latino and Black engineers in computer science and engineering in the USA.
---

## SHPE Stevens Chapter Website
The Society of Hispanic Professional Engineers (SHPE) Stevens Chapter website, serving as a hub for Latino engineers and CS students at Stevens Institute of Technology.

- **Live Site:** [https://shpestevens.vercel.app/](#)
- **Contributors:**
  - [Matias Freire](https://github.com/MatiasPF1)

---

## ColorStack Stevens Chapter Website
The ColorStack Stevens Chapter website, supporting Black and Latino students pursuing careers in computer science and engineering.
- **Live Site:** [https://colorstack-stevens.vercel.app/](#)
- **Contributors:**
  - [Matias Freire](https://github.com/MatiasPF1)

## Getting Started

Each project lives in its own folder:

- `SHPE-Stevens-Chapter/` — SHPE website (Next.js)
- `Colorstack-StevensChapter/` — ColorStack website (Next.js)

Navigate into either folder and run:

```bash
npm install
npm run dev
```
